-----------------------------------------------------------------------
      フィールド分離・結合 フィルタプラグイン ver0.02  by MakKi
-----------------------------------------------------------------------

【機能】

  フィールドを分離・結合します。
  インターレース未対応のフィルタで、フィールドごとに処理したい時に有効です。

【インストール】

  field_sp.auf、field_wv.auf を AviUtl の認識可能なフォルダにコピーしてください。
  単体では使用できません。

【使用方法】

  設定→フィルタ順序の設定→ビデオフィルタ順序の設定 で、フィールド分離、
  フィールド結合の間にフィールド単位で処理したいフィルタをはさんで下さい。
  ただし、リサイズ・クリッピング等、画像サイズを変更するフィルタをはさむと
  分離モードによっては画像が崩れる場合があります。
  また、フィールド分離よりフィールド結合の順序を先にしても画像が崩れる場合があります。

  ○パラメータ

    ・間隔                  : トップフィールド画像とボトムフィールド画像の間の余白の幅を設定します。
    ・Mode                  : 分離モードを指定します。（各モードの説明は下記参照）
    ・間隔を接するピクセルの色で埋める
                            : 間隔を接するピクセルの色で埋めます。（offでは黒で埋める）
    ・トップとボトムの配置を逆にする
                            : 分離状態での各フィールドの配置を逆にします。
    ・分離表示              : フィールドを分離した状態を表示します。（フィールドを結合しない）

    ※フィールド結合フィルタには設定ウィンドウはありません。

  ○分離モード
    Mode 1 : 分離した画像を縦に並べます。
    Mode 2 : 分離した画像を横に並べます。縦が1/2、横が2倍になります。
             分離前の縦ピクセル数が奇数の場合、最下フィールドが加えられ偶数になります。
    Mode 3 : 分離した画像を1フレームとして交互に並べます。
             縦が1/2になり、総フレーム数が２倍になります。
             分離前の縦ピクセル数が奇数の場合、最下フィールドが加えられ偶数になります。             
             時間軸系のフィルタでフィールドが混合してしまいます。
    Mode 4 : 分離した画像を1フレームとして、前半と後半にフィールドを分けて並べます。
             縦が1/2になり、総フレーム数が２倍になります。
             分離前の縦ピクセル数が奇数の場合、最下フィールドが加えられ偶数になります。             
             Mode3と違い、時間軸系のフィルタでもフィールドが混合しません。

【既知の問題】

    ・ぼかしやシャープの効果が強まる
        分離状態では画像の縦サイズが半分になっているため、フィルタによっては効果が強まります。
        今後の更新で対処する機能を付けるかもしれません。
    ・Mode3、Mode4の処理が重い
        仕様です。他のモードを使うか、諦めてください。
